## 数据集选择
+ kaggle 房价预测数据
+ kaggle 泰坦尼克号生存问题
+ spambase 邮件分类数据集
+ dota2Datas Dota2游戏胜负预测
+ kaggle mnist手写题识别
+ wine_quality 白葡萄酒质量数据集
+ lendingclub 信贷违约数据集
